# Laboratorio #1- Matriz 100.000 x 100.000

Daniel David Durango Durango

Este codigo genera la matriz de 100.000 x 100.000 con valores binarios (0 y 1) 
y la guarda en el disco duro en formato binario comprimido usando el `numpy.packbits`.

## Tamaño del archivo que se generó

- Almacenamiento por bit (1 bit por número)
- Tamaño aproximado de ~1.25 GB

## Tiempo de ejecución
Me tardó alrededor de 5 minutos, supongo que es porque lo estoy haciendo en una laptop no tan potente.
